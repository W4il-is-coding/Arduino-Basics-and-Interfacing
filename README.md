# Arduino Basics and Interfacing

This repository contains a collection of basic Arduino programs created to understand
fundamental embedded systems concepts and hardware interfacing.

## Topics Covered
- LED control and digital I/O
- Sensor interfacing
- Motor control
- Timing and delay-based programs

## Purpose
These programs were written for practice and learning Arduino programming and
microcontroller basics.

## Tools Used
- Arduino UNO
- Arduino IDE
- Basic electronic components
